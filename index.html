تعلم HTML

ما هي HTML؟

HTML (لغة ترميز النص التشعبي) هي العمود الفقري لأي صفحة ويب. تستخدم لإنشاء هيكل المحتوى وتنظيمه.

العناصر الأساسية

<!DOCTYPE html>
<html>
<head>
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>عنوان رئيسي</h1>
    <p>فقرة نصية</p>
    <a href="https://example.com">رابط</a>
    <img src="image.jpg" alt="وصف الصورة">
</body>
</html>

تعلم CSS

ما هي CSS؟

CSS (أوراق الأنماط المتتالية) هي لغة تستخدم لتنسيق وتهيئة مظهر عناصر HTML.

أساسيات CSS

/* تغيير لون النص لجميع العناوين */
h1, h2, h3 {
    color: #3498db;
}

/* تنسيق الفقرات */
p {
    font-size: 16px;
    line-height: 1.6;
    margin-bottom: 20px;
}

/* تنسيق الروابط */
a {
    text-decoration: none;
    color: #e74c3c;
}

/* تنسيق الصفحة */
body {
    font-family: Arial, sans-serif;
    background-color: #f9f9f9;
}

تعلم JavaScript

ما هي JavaScript؟

JavaScript هي لغة برمجة تستخدم لجعل صفحات الويب تفاعلية وديناميكية.

أساسيات JavaScript

// تعريف متغير
let message = "مرحبًا بالعالم!";

// طباعة في وحدة التحكم
console.log(message);

// دالة بسيطة
function greet(name) {
    alert("مرحبًا " + name + "!");
}

// حدث عند النقر على زر
document.getElementById("myButton").addEventListener("click", function() {
    greet("زائر");
});

// مصفوفة
let colors = ["أحمر", "أخضر", "أزرق"];

// حلقة تكرار
for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}
